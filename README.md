# Ebola-DDoS

INSTALL On Linux
```
apt-get update
apt-get install python3 python3-pip
git clone https://github.com/413xPr06605/Ebola-DDoS
cd Ebola-DDoS
```

Linux Commands:
```
python3 ebola.py
```
